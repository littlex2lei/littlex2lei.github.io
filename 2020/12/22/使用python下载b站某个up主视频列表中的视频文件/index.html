使用python下载b站某个up主视频列表中的视频文件

1.下载清单

1.1 在b站上进行初步搜索

1.2 在up主主页搜索相关内容

1.3 用firefox 分析视频链接在哪

按F12打开网络页面分析的工具
点击网络标签，然后按F5刷新，在原网页更新内容时，页面分析工具可以得到当前网页不同部分的数据来源
在下方过滤器中 输入api.bilibili.com，选取与数据调用接口相关的内容。
一个个点开看之后，发现与我关心的数据调用的链接为

https://api.bilibili.com/x/space/arc/search?mid=454145207&ps=75&tid=0&pn=1&keyword=abaqus&order=pubdate&jsonp=jsonp

从这个网页中可以获取我们所想要下载的视频的关键信息 bvid

另外这个网址有点长，只有复制粘贴到地址栏才能看到想要的信息

1.4 在python中读取不同视频的bvid

将b站的视频分两类：

  • 单独的视频
    对于单独视频，其网页链接格式为 https://www.bilibili.com/video/ + 对应视频的bvid
    有bvid信息就可以直接用youget下载
  • 包含播放列表 playlist的视频
    对于播放列表, you-get 可以在参数中加入 -playlist将所有视频全部下载， 但是目前暂时没有单独下载某一个视频的解决方案。

2. 使用you-get 下载视频

关于you-get的详细资料参见
https://you-get.org/
https://pypi.org/project/you-get/
https://github.com/soimort/you-get

2.1 you-get的使用方式简述

you-get -o 目标文件夹 下载网址

you-get -o 目标文件夹 下载网址 –playlist

you-get -i https://www.bilibili.com/video/BV1r64y1f7ra

-i参数可以分析可下载文件的名称以及格式，这里主要也是看格式 选择一个清晰度再下载

音轨和视频视频的合并可参见下文
https://www.cnblogs.com/cyx-b/p/13833369.html
