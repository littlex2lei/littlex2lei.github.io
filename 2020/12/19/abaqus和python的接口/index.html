abaqus和python的接口

本篇主要记录下 abaqus和python交互的主要途径。

1.Abaqus脚本接口功能

Abaqus脚本接口可以实现下列功能：
1） 创建、修改Abaqus模型中的属性（例如，部件、材料、荷载和分析步等）。
2） 创建、修改和提交分析作业。
3） 查看分析结果。

2.Abaqus中不同类型的脚本接口

Python是abaqus的标准编程语言。除了命令行接口之外，Abaqus还提供了下列脚本接口；

1）环境文件abaqus_v6.env中使用的是Python语句,详细介绍请见《Abaqus Scripting User’s Manual》6.9 节 “ Using Abaqus Scripting Interface commands in your environment file
2） INP文件中* PARAMETER数据行中的参数定义使用的是Python语句。关于* PA- RAMETER用法的详细介绍，请参见Abaqus 6. 10帮助手册《Abaqus Keywords Reference Man- ual》o
3） 运行参数分析需要编写和执行Python脚本(.psf)
4） Abaqus/CAE将所有的命令记录在扩展名为.rpy的文件中，这类文件中的所有命令均为Python访句。
5） 在Abaqus/CAE的File菜单下选择Run Script,可以直接运行脚本。
6） 在Abaqus/CAE的File菜单下选择Macro Manager,可以录制宏。

abaqus中脚本运行方法

  • 启动Abaqus/CAE的同时运行脚本
    成熟的回测程序减少了清理数据并进行迭代等策略测试中所必须的繁琐过程。它具有可用于各种数据源的内置函数，保证数据导入更加容易。

  • 不启动Abaqus/CAE而直接运行脚本
    策略参数的调整有时可能带来同类策略是否盈利的差别。运行回测后，只需更改几行代码即可轻松完成优化。

  • 从启动屏幕（startup screen）运行脚本
    Python的绘图工具并不是十分友好，尤其是第一次使用时。在Backtrader中可用少量代码创建复杂的图表。
